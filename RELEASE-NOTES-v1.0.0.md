# Groundwork v1.0.0 🚀

The first public release of **Groundwork** — an enterprise-grade knowledge base + agent library that makes AI "vibe coding" produce production-ready software, in any AI coding tool.

## What's inside

- **`standards-kb/`** — 20 non-functional standard domains + a master release gate (the bar every feature must meet)
- **15 feature blueprints** — report builder, RBAC, subscription & billing, workflow & approval engine, churn analytics, and more
- **`admin-kb/`** — a full Admin Area (login history, security center, log explorer, API analytics, consent/DSR, retention…)
- **19 expert agents** — Solutions Architect, Security Officer, QA, DevOps, Compliance Officer, Feature Architect, API Platform & MCP engineers, and more
- **Design system + 36-component catalog**, **performance**, **events (async patterns)**, **SRE runbooks**, **compliance (GDPR/DPDP/SOC2/ISO/PCI/HIPAA)**, **threat models**, **FinOps**, **OpenAPI** and **MCP** knowledge bases
- **Templates + a 6-step prompt playbook**, and ready-made rules files for Claude Code, Cursor, Copilot, Windsurf, and Cline

## Install

```bash
npx degit ai-code-factory-hub/ai-coding-standards groundwork
# then copy the rules file for your tool — see README.md and TUTORIAL.md
```
